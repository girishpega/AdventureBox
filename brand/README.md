## What This Is

This is the visual soul of AdventureBox — the document we read before we write another line of code, generate another illustration, or design another screen.

AdventureBox is not an app that contains stories.  
It is a **children's reading brand** — the next shelf a parent reaches for after Disney Storybook, Kindle Kids, and the worn copy of *The Gruffalo*.

## Brand Essence

| Dimension | AdventureBox |
|-----------|--------------|
| **Promise** | Wonder you can hold |
| **Feeling** | Cozy magic — not chaos, not lecture |
| **Audience** | Ages 4–6 first. Parents as co-readers. |
| **Enemy** | Screen fatigue, loud apps, hollow gamification |
| **Hero** | The moment a child says *"Again!"* |

---

## The Five Faces of AdventureBox

Every AdventureBox story belongs to one of five character worlds — but all five characters live in the same universe and may appear together on covers, end screens, and future "AdventureBox Friends" content.

```
    ✦ Pip          ♥ Ellie         ★ Nova
   The Light      The Heart       The Wonder

    ☺ Finn         📖 Sage
   The Play    The Storyteller
```

## One Line Brand Test

Show a parent the mood board and one book cover.  
If they say *"I'd read this to my kid tonight"* — we build.  
If they say *"Is this an app?"* — we revise.
