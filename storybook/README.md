## Story Logline

A gentle baby elephant hears a tiny peep in a sunlit meadow — and discovers that the best friends are sometimes hiding in the silliest places.

## Production Workflow

1. Read **ART_DIRECTION.md** before generating any image.
2. Generate illustrations in order (Page 1 → Page 5) using prompts in each PAGE file.
3. Review all five images side-by-side for character consistency before approval.
4. Drop approved images into the existing AdventureBox reader — no new code required.
